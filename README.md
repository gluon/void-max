## Installation

### Option 1 — Download a release (recommended)

1. Go to the [Releases page](../../releases) and download the latest `void-x.y.z.zip`
2. Unzip — you get a `void/` folder
3. Move the `void/` folder into your Max packages folder:
   - **macOS**: `~/Documents/Max 9/Packages/`
   - **Windows**: `Documents\Max 9\Packages\`
4. Restart Max — all externals appear in the autocomplete

### Option 2 — Build from source

Clone this repo, then build each external from its `source/<name>/` folder using CMake against the [Max SDK base](https://github.com/Cycling74/max-sdk-base).
